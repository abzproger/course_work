Кредитный калькулятор
Это простое приложение, разработанное с использованием библиотеки PyQt6 и Python. Оно предоставляет пользователю возможность рассчитать количество месяцев, необходимых для достижения определенной целевой суммы при заданной процентной ставке на вкладе.

Установка
Для запуска программы вам понадобится Python версии 3.x и библиотека PyQt6. Вы можете установить PyQt6 с помощью pip:

bash
pip install PyQt6
Как использовать
Запустите программу, выполнив файл main.py.
Введите начальный вклад, процентную ставку и целевую сумму в соответствующие поля ввода.
Нажмите кнопку "Вычислить".
Результат расчета будет отображен в окне программы.
Описание функционала
Начальный вклад: Сумма денег, которую вы вносите в начале периода.
Процентная ставка: Годовая процентная ставка, начисляемая на ваш вклад.
Целевая сумма: Сумма денег, которую вы хотите достичь.
Пример использования
Предположим, вы внесли начальный вклад в размере 1000 рублей с процентной ставкой 5% и целью накопить 2000 рублей. После нажатия кнопки "Вычислить" программа сообщит вам, что для достижения цели потребуется определенное количество месяцев, а также итоговую сумму вклада.

Примечания
Программа обрабатывает случай, когда целевая сумма меньше или равна начальному вкладу, и сообщает об ошибке.
